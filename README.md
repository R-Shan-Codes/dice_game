# 🎲 Dice Roll Game

A simple two-player dice game built in Python using a modular structure and menu-driven system.

---

## 📌 Features

* 🎲 Random dice roll using `random` module
* 👥 Two-player gameplay
* 🧮 Score tracking system
* 📋 Menu-driven program (`while True` loop)
* 📦 Clean modular structure (services, utils)

---

## 🧠 How It Works

* Each player rolls a dice (1–6)
* Player with higher number wins the round
* Score updates after every round
* Option to view or reset score anytime

---

dice-game/

## ▶️ How to Run

1. Open terminal in project folder
2. Run:

```
python main.py
```

---

## 🕹️ Menu Options

* 1 → Play Round
* 2 → Show Score
* 3 → Reset Score
* 4 → Exit
